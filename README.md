# ai article summarizer — privacy policy

this repository hosts the privacy policy for the [ai article summarizer](https://chrome.google.com/webstore) chrome extension.

## about the extension

ai article summarizer is a chrome extension that lets you instantly summarize any article or webpage using your own google gemini api key. choose between brief, detailed, or bullet point summaries with a clean, minimal interface.

## privacy

this extension does not collect, store, or transmit any personal data. your gemini api key is stored locally using chrome's built-in sync storage and is only used to make requests directly to google's gemini api on your behalf.
