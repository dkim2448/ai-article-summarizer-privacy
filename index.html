privacy policy — ai article summarizer

last updated: may 2026

overview

ai article summarizer is a chrome extension that summarizes web articles using the google gemini api. we are committed to protecting your privacy.

data we collect

the only data this extension handles is the text content of web pages you choose to summarize, and your gemini api key. no personal data is collected.

how your data is used

page text is sent directly to the google gemini api to generate a summary. it is not stored, logged, or shared with anyone. your gemini api key is stored locally using chrome's built-in sync storage and is used solely to authenticate requests to the gemini api.

data sharing

we do not sell, trade, or transfer any data to third parties. page content is sent only to google's gemini api as part of the summarization process.

third party services

this extension uses the google gemini api. please refer to google's privacy policy at policies.google.com/privacy for information on how google handles data.

contact

if you have any questions about this privacy policy, please open an issue on the extension's github repository.
